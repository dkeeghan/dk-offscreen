# dk-offscreen

A component for navigation drawer style navigation.

## Features:

- `dk-offscreen-toggle` - A button that keeps in sync with the navigation drawer to ensure that the appropriate `aria-` states and classes are applied
- `dk-offscreen` - An navigation drawer that will show and hide based on user input.

## How to use

### Installation

```bash
npm install dk-offscreen
```

```js
import 'dk-offscreen/dk-offscreen.js';
```
